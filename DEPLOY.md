# アプリへのアクセス・デプロイ方法

作成した「アニメ名言サーチ」をPCやスマホで見るには、主に2つの方法があります。

## 1. 同じWi-Fi内のスマホ・PCからアクセスする（今すぐ試せます）

PCとスマホが同じWi-Fi（ネットワーク）に接続されている場合、PCのIPアドレスを使ってアクセスできます。

1.  PCで `npm run dev` が実行されていることを確認してください。
2.  スマホのブラウザ（SafariやChrome）を開きます。
3.  アドレスバーに以下を入力してアクセスしてください。

    `http://192.168.0.11:3000`

    ※ あなたのPCの現在のIPアドレス: `192.168.0.11` (変動する可能性があります)

**注意**: PCのファイアウォール設定によっては接続できない場合があります。その場合は「パブリックネットワーク」と「プライベートネットワーク」の通信許可を確認してください。

---

## 2. インターネット上に公開する (Vercel)

世界中のどこからでもアクセスできるようにするには、Next.jsの開発元である **Vercel** へのデプロイが最も簡単でおすすめです。

### 手順

1.  **GitHubにリポジトリを作成**
    - このプロジェクトのコードをGitHubにプッシュします。

2.  **Vercelアカウント作成**
    - [Vercel](https://vercel.com) にアクセスし、GitHubアカウントでログインします。

3.  **プロジェクトのインポート**
    - Vercelのダッシュボードで「Add New...」 -> 「Project」を選択。
    - 先ほどプッシュしたGitHubリポジトリを選択して「Import」をクリック。

4.  **環境変数の設定**
    - **Environment Variables** の設定項目で、以下を追加します。
        - Key: `NEXT_PUBLIC_GEMINI_API_KEY`
        - Value: `（あなたのGemini APIキー）`

5.  **デプロイ実行**
    - 「Deploy」ボタンをクリックすると、自動的にビルドが始まり、数分で公開されます。
    - 公開用URL（例: `anime-quotes.vercel.app`）が発行されます。

これで、友達にもシェアできるようになります！
